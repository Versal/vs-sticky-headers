Classical logic

Classical logic describes statements that can be either true or false. Operations of classical logic are "not", "and", "or", "implies". Statements are derived from axioms or from previously derived statements ("theorems").

Intuitionistic logic

Intuitionistic logic logic describes statements that can be true, false, or not yet derived. Operations of intuitionistic logic are the same as in classical logic - "not", "and", "or", "implies". However, while proving theorems one cannot assume that every statement is necessarily either true or false.

Modal logic

The simplest version of modal logic adds to classical logic the operators "possibly true" and "necessarily true". More generally, modal logic introduces the idea that a statement can be true "under certain circumstances" and false otherwise. The notion of "circumstances" is made formal by introducing labels. Something is "necessarily true" if it is true under "all possible circumstances".

Temporal logic

Temporal logic is a particular case of modal logic where the "different circumstances" are just different moments of time. Time is discrete, and all available moments of time are numbered 0, 1, 2, ... A given statement can be true at one moment and false at another moment.
